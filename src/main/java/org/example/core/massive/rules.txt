
Массивы в Java — это коллекции однотипных элементов (то есть элементов с одним и тем же типом данных),
хранящихся в непрерывной области памяти. Они являются объектами и, следовательно, хранятся в куче.
Каждому элементу массива можно доступиться через индекс, который начинается с нуля.

Особенности массивов в Java
Фиксированная длина: Как только массив создан, его размер не может быть изменён.
Типизированность: Все элементы массива должны иметь один и тот же тип данных.
Производительность: Доступ к элементам массива происходит очень быстро, так как элементы хранятся в памяти последовательно.
Создание массива
Массивы можно создавать различными способами. Ниже представлены примеры объявления, создания и инициализации массивов.

Объявление массива:

int[] myArray;
Создание массива:

myArray = new int[10]; // массив из 10 целых чисел
Инициализация массива:

myArray[0] = 100; // присваивание значения первому элементу
Объявление, создание и инициализация в одной строке:

int[] myArray = {1, 2, 3, 4, 5};
Работа с массивами
Вы можете выполнять различные операции с массивами, например, перебирать элементы, сортировать их, искать определённые элементы и т.д.

Пример перебора элементов массива с помощью цикла:

for (int i = 0; i < myArray.length; i++) {
    System.out.println(myArray[i]);
}
