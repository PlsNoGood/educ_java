Циклы в Java играют ключевую роль в управлении потоком выполнения программы, позволяя повторять набор инструкций многократно до тех пор,
пока не будет выполнено заданное условие. В Java есть три основных типа циклов: `for`, `while` и `do-while`. Рассмотрим каждый из них подробнее.

### Цикл for

Цикл `for` идеально подходит, когда заранее известно количество итераций. Он имеет следующий синтаксис:

for (инициализация; условие; шаг) {
    // блок кода для выполнения
}

- **Инициализация** — начальная настройка переменной, например, `int i = 0`.
- **Условие** — определяет, будет ли цикл продолжаться. Цикл выполняется, пока условие истинно.
- **Шаг** — инструкция, выполняемая после каждой итерации цикла, обычно это инкремент или декремент, например, `i++`.

#### Пример:

for (int i = 0; i < 5; i++) {
    System.out.println("i равно " + i);
}


### Цикл while

Цикл `while` выполняется, пока истинно заданное условие. Он полезен, когда количество итераций не известно заранее.


while (условие) {
    // блок кода для выполнения
}


#### Пример:

int i = 0;
while (i < 5) {
    System.out.println("i равно " + i);
    i++;
}


### Цикл do-while

Цикл `do-while` похож на цикл `while`, но он гарантирует, что блок кода выполнится хотя бы один раз, поскольку проверка
условия происходит после выполнения блока кода.


do {
    // блок кода для выполнения
} while (условие);


#### Пример:

int i = 0;
do {
    System.out.println("i равно " + i);
    i++;
} while (i < 5);


### Распространенные использования циклов:

- **Обработка коллекций и массивов**: Чтобы пройти по всем элементам массива или коллекции.
- **Чтение данных до маркера окончания**: Например, чтение из файла до достижения конца файла.
- **Ожидание события**: Например, ожидание, пока пользователь не введет правильные данные.

### Примеры задач, решаемых с помощью циклов:

1. **Суммирование чисел**: Например, суммирование всех чисел от 1 до N.
2. **Нахождение максимального/минимального значения**: Поиск максимального или минимального значения в массиве.
3. **Решето Эратосфена**: Алгоритм для нахождения всех простых чисел до заданного числа N.

Циклы — это мощный инструмент в языках программирования, позволяющий сделать код более эффективным и компактным.
Главное — правильно выбирать тип цикла в зависимости от задачи и условий её выполнения.
